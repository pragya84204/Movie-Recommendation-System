# Movie-Recommendation-System
Movie Recommendation System using Collaborative Filtering

This project builds a movie recommendation system using collaborative filtering.  
The system recommends movies to users based on similarity between movie ratings.

## Dataset
The project uses the MovieLens dataset which contains movie ratings given by users.

Dataset files:
- movies.csv
- ratings.csv

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Data loading and preprocessing
2. Exploratory Data Analysis (EDA)
3. Creating user-movie matrix
4. Calculating movie similarity using correlation
5. Generating movie recommendations

## Example Recommendation

Input Movie: Toy Story (1995)

Recommended Movies:
- Toy Story 2
- Bug's Life
- Babe
